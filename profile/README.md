> [!IMPORTANT]
> **Modernize your legacy. Scale without rewriting.**
>
> Yii2 Extensions helps teams keep proven Yii2 applications productive while adopting PSR interoperability,
> worker runtimes, static analysis, modern frontend adapters, and repeatable scaffolding.

## Ecosystem map

<details open>
<summary><strong>Runtime, PSR, and worker mode</strong></summary>

- [PSR Bridge](https://github.com/yii2-extensions/psr-bridge): PSR-7 and PSR-17 integration with worker lifecycle support.
- [RoadRunner](https://github.com/yii2-extensions/road-runner): high-performance HTTP server integration.
- [FrankenPHP](https://github.com/yii2-extensions/franken-php): FrankenPHP worker integration for modern PHP deployments.
- [Worker Debug](https://github.com/yii2-extensions/worker-debug): debug toolbar support for PSR Bridge based workers.

</details>

<details open>
<summary><strong>Application templates and scaffolding</strong></summary>

- [App Base](https://github.com/yii2-extensions/app-base): shared scaffold provider with entrypoints, authentication, RBAC, mail, migrations, and quality tooling.
- [App Basic](https://github.com/yii2-extensions/app-basic): modern Bootstrap 5 application template.
- [App Advanced](https://github.com/yii2-extensions/app-advanced): advanced Yii2 application template.
- [Scaffold](https://github.com/yii2-extensions/scaffold): Composer plugin for composable Yii2 application scaffolds.
- [Template](https://github.com/yii2-extensions/template): extension template with PHPUnit, PHPStan, Infection, ECS, GitHub Actions, and Codecov.

</details>

<details>
<summary><strong>Inertia, frontend, UI, and widgets</strong></summary>

- [Inertia](https://github.com/yii2-extensions/inertia), [Inertia React](https://github.com/yii2-extensions/inertia-react), and [Inertia Vue](https://github.com/yii2-extensions/inertia-vue): modern Inertia.js adapters for Yii2.
- [App Inertia React](https://github.com/yii2-extensions/app-inertia-react) and [App Inertia Vue](https://github.com/yii2-extensions/app-inertia-vue): complete reference applications.
- [Frontend jQuery](https://github.com/yii2-extensions/frontend-jquery): traditional Yii2 frontend package.
- [Bootstrap 5](https://github.com/yii2-extensions/bootstrap5), [Asset Bootstrap 5](https://github.com/yii2-extensions/asset-bootstrap5), [Asset FontAwesome](https://github.com/yii2-extensions/asset-fontawesome), [Asset Popper](https://github.com/yii2-extensions/asset-popper), and [Asset Bootbox](https://github.com/yii2-extensions/asset-bootbox): reusable UI assets.
- [FilePond](https://github.com/yii2-extensions/filepond), [Select2](https://github.com/yii2-extensions/select2), [Selectize](https://github.com/yii2-extensions/selectize), [Dynamic Form](https://github.com/yii2-extensions/dynamicform), [Calendar](https://github.com/yii2-extensions/calendar), [DateTime Picker](https://github.com/yii2-extensions/datetime-picker), [Time Picker](https://github.com/yii2-extensions/time-picker), and [Summernote](https://github.com/yii2-extensions/summernote): widgets and integrations for production interfaces.

</details>

<details>
<summary><strong>Developer tooling, core packages, and examples</strong></summary>

- [Debug](https://github.com/yii2-extensions/debug): debugger panels for Yii2 applications.
- [PHPStan](https://github.com/yii2-extensions/phpstan): enhanced static analysis for Yii2.
- [Docker Images](https://github.com/yii2-extensions/docker-images): Docker images for repeatable development and delivery.
- [Core Library](https://github.com/yii2-extensions/core-library): shared library code used across packages.
- [User](https://github.com/yii2-extensions/user), [Locale URLs](https://github.com/yii2-extensions/localeurls), [Nested Sets Behavior](https://github.com/yii2-extensions/nested-sets-behavior), [PostGIS](https://github.com/yii2-extensions/postgis), [Sentry](https://github.com/yii2-extensions/sentry), and [Imagine](https://github.com/yii2-extensions/imagine): focused Yii2 extensions.
- [Blog](https://github.com/yii2-extensions/blog), [Demo](https://github.com/yii2-extensions/demo), [Demo Blog](https://github.com/yii2-extensions/demo-blog), [Cookbook](https://github.com/yii2-extensions/cookbook), [Maintained Third-Party](https://github.com/yii2-extensions/maintained-third-party), and [Extensions](https://github.com/yii2-extensions/extensions): examples, guides, and community catalogs.

</details>
