## About

**Yii2 Extensions** maintains production-oriented packages that help teams keep Yii2 applications aligned with the
modern PHP ecosystem. The organization focuses on PSR interoperability, async-capable runtimes, reusable application
foundations, strict static analysis, and practical upgrade paths for existing codebases.

## Featured ecosystem

### Runtime and interoperability

- [PSR Bridge](https://github.com/yii2-extensions/psr-bridge): PSR-7 and PSR-17 integration for Yii2, including
  worker lifecycle support.
- [RoadRunner](https://github.com/yii2-extensions/road-runner): high-performance HTTP server integration for
  long-running Yii2 applications.
- [FrankenPHP](https://github.com/yii2-extensions/franken-php): FrankenPHP worker integration for modern PHP
  deployments.
- [Worker Debug](https://github.com/yii2-extensions/worker-debug): debug toolbar support for PSR Bridge based workers.

### Application scaffolds

- [App Base](https://github.com/yii2-extensions/app-base): shared scaffold provider with entrypoints, authentication,
  RBAC, mail, migrations, and quality tooling.
- [App Basic](https://github.com/yii2-extensions/app-basic): modern Bootstrap 5 application template.
- [App Advanced](https://github.com/yii2-extensions/app-advanced): advanced Yii2 application template.
- [App Inertia React](https://github.com/yii2-extensions/app-inertia-react): React 19, shadcn/ui, Radix, Tailwind CSS,
  and Vite reference application.
- [App Inertia Vue](https://github.com/yii2-extensions/app-inertia-vue): Vue 3, Flowbite, Tailwind CSS, and Vite
  reference application.

### Inertia and frontend

- [Inertia](https://github.com/yii2-extensions/inertia): server-side adapter for Yii2 page rendering, shared props,
  partial reloads, flash data, errors, and asset versions.
- [Inertia React](https://github.com/yii2-extensions/inertia-react): React adapter with Vite integration and bootstrap
  wiring.
- [Inertia Vue](https://github.com/yii2-extensions/inertia-vue): Vue adapter with Vite integration and bootstrap
  wiring.
- [Frontend jQuery](https://github.com/yii2-extensions/frontend-jquery): frontend package for traditional Yii2
  applications.

### Developer tooling

- [PHPStan](https://github.com/yii2-extensions/phpstan): enhanced static analysis for Yii2 applications.
- [Debug](https://github.com/yii2-extensions/debug): debugger panels for requests, logs, queries, profiling, timeline,
  queues, mail, events, assets, and user switching.
- [Scaffold](https://github.com/yii2-extensions/scaffold): Composer plugin for composable Yii2 application scaffolds.
- [Template](https://github.com/yii2-extensions/template): extension template with PHPUnit, PHPStan, Infection, ECS,
  GitHub Actions, and Codecov.
- [Docker Images](https://github.com/yii2-extensions/docker-images): Docker images for repeatable Yii2 development and
  delivery.

### UI, widgets, and assets

- [Bootstrap 5](https://github.com/yii2-extensions/bootstrap5), [Asset Bootstrap 5](https://github.com/yii2-extensions/asset-bootstrap5),
  [Asset FontAwesome](https://github.com/yii2-extensions/asset-fontawesome), [Asset Popper](https://github.com/yii2-extensions/asset-popper),
  and [Asset Bootbox](https://github.com/yii2-extensions/asset-bootbox): reusable UI assets for Yii2.
- [FilePond](https://github.com/yii2-extensions/filepond), [Select2](https://github.com/yii2-extensions/select2),
  [Selectize](https://github.com/yii2-extensions/selectize), [Dynamic Form](https://github.com/yii2-extensions/dynamicform),
  [Calendar](https://github.com/yii2-extensions/calendar), [DateTime Picker](https://github.com/yii2-extensions/datetime-picker),
  [Time Picker](https://github.com/yii2-extensions/time-picker), and [Summernote](https://github.com/yii2-extensions/summernote):
  widgets and integrations for production Yii2 interfaces.
