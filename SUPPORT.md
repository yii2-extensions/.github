# Support

## Getting Help

- **Bug reports**: [Open a bug report](https://github.com/yii2-extensions/.github/issues/new?template=bug-report.yml).
- **Feature requests**: [Open a feature request](https://github.com/yii2-extensions/.github/issues/new?template=feature-request.yml).
- **Security vulnerabilities**: See [SECURITY.md](SECURITY.md). Do not open public issues.
